Import Backlog Updates From Excel

Required columns: Roll Number, Semester Number, New backlog.

For roll numbers that start with 0, format the column as Text in Excel before entering values. If Excel stores them as numbers, leading zeros are dropped and the roll will not match the database.

Each row updates that semester backlog to the new value. Cleared backlog (old - new) is added to dead backlog automatically.
